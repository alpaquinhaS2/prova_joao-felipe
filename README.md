# prova_joao-felipe
atividade avaliativa final da matéria web front com recomendações de filmes/games 
Este projeto é um mini-site estático em HTML e CSS chamado Guia de Recomendações do Alpaca (eu, joao felipe), criado com o objetivo de recomendar meus filmes e jogos favoritos.
A página apresenta textos, imagens, listas, links externos oficiais, além de uma tabela organizada contendo informações como categoria, ano de lançamento e notas de IMDb/Metacritic.
O site foi desenvolvido para praticar os fundamentos de HTML e CSS, incluindo estruturação, estilização, uso de listas, tabelas, âncoras e organização de conteúdo.
